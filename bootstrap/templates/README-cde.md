# ASAP CRN Cloud Common Data Elements (CDE)

This repository contains the Common Data Elements used across ASAP CRN datasets and collections.

## Structure

- `cde.json`: JSON index containing all Common Data Elements with their definitions and references

## CDE Metadata Schema

```json
{
  "cde_version": "CDE_001",
  "date": "July 03, 2025",
  "status": "active|deprecated"
}
```



## Management

This repository is automatically managed by the [cloud-orchestration](https://github.com/ASAP-CRN/cloud-orchestration) system. Manual changes should be avoided.

Common Data Elements ensure consistency across datasets. For CDE submissions or updates, please use the orchestration system or contact the ASAP CRN team.
